Interactive Website Project

Project Overview:

  • Describe your application and its purpose: The application will store a student's courses and will notify users via email of upcoming/missed assignments. The purpose of this app is to be an extra help for students in completing in time.
  • Intended users of the website: Intended users of the website are college students.
  • Overview of the content: the landing page will list upcoming events/due dates for the user's assignments; upon missed or important upcoming events, the user will receive a notification via email; the presented information will be sectioned into categories based on urgency
